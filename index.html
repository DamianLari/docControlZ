ControlZ Documentation

Introduction

Bienvenue dans le guide utilisateur de ControlZ, votre compagnon de confiance pour la sauvegarde de données personnelles et professionnelles. ControlZ a été conçu pour offrir une solution de sauvegarde facile à utiliser tout en étant puissante et flexible.

À propos de ControlZ

ControlZ est une application dédiée à la sauvegarde manuelle de données, développée pour apporter une solution simple et directe aux besoins de sauvegarde quotidiens. Que vous cherchiez à protéger vos documents académiques, à sécuriser vos projets personnels ou à sauvegarder tout contenu numérique important, ControlZ offre une approche intuitive et conviviale.

Conçue pour être utilisée sans connaissances techniques avancées, ControlZ se positionne comme l'outil idéal pour ceux qui souhaitent contrôler manuellement leurs sauvegardes sans complication.

Objectif de l'application

L'objectif de ControlZ est de rendre la sauvegarde des données accessible et gérable pour tout le monde, sans avoir besoin de compétences techniques approfondies. L'application vise à offrir une tranquillité d'esprit en permettant aux utilisateurs de créer facilement des sauvegardes complètes ou différentielles de leurs données importantes.

Fonctionnalités principales

ControlZ est équipé des fonctionnalités suivantes pour une gestion optimale de vos données :

  • Sauvegardes personnalisées : Créez des sauvegardes adaptées à vos besoins en quelques étapes simples.
  • Modification aisée : Mettez à jour vos paramètres de sauvegarde en toute simplicité.
  • Suppression de sauvegardes : Éliminez les anciens travaux de sauvegarde que vous n'utilisez plus pour maintenir votre espace de travail organisé.
  • Exécution manuelle : Démarrez des sauvegardes individuelles quand vous le souhaitez pour une mise à jour immédiate de vos données.
  • Exécution séquentielle : Lancez plusieurs travaux de sauvegarde les uns après les autres pour une mise à jour en série de vos fichiers.
  • Choix de langue : Interagissez avec l'application dans la langue qui vous est la plus confortable.
  • Fermeture de l'application : Après avoir terminé vos sauvegardes, fermez l'application en toute sérénité.

Dans les sections qui suivent, vous trouverez des instructions détaillées pour chaque fonctionnalité, vous permettant de débuter rapidement et facilement avec ControlZ.

Démarrage

Avant de pouvoir profiter pleinement de ControlZ, quelques étapes préliminaires sont nécessaires pour préparer votre environnement de travail. Ce chapitre vous guide à travers le processus d'installation et de configuration initial pour que vous puissiez commencer à utiliser l'application sans délai.

Prérequis système

Pour utiliser ControlZ, votre système doit satisfaire les prérequis suivants :

  • Un ordinateur sous Windows 10/11.
  • Une connexion internet active pour le téléchargement des composants nécessaires.
  • Des droits d'administrateur sur votre machine pour l'installation de logiciels.

Installation de .NET 7

ControlZ est construit sur la plateforme .NET 7, qui doit être installée sur votre système pour que l'application fonctionne correctement.

  1. Télécharger .NET 7 :
  • Rendez-vous sur la page de téléchargement officielle de .NET.
  • Sélectionnez le SDK .NET correspondant à votre système d'exploitation.
  • Téléchargez et lancez l'installeur.
  1. Suivre les instructions d'installation :
  • Exécutez le programme d'installation et suivez les instructions à l'écran pour installer .NET 7 sur votre système.
  • Une fois l'installation terminée, vous pouvez vérifier l'installation en ouvrant une invite de commande ou un terminal et en tapant ‘dotnet –version’. Vous devriez voir le numéro de version de .NET 7 s'afficher.

Téléchargement et installation de ControlZ

Avec .NET 7 installé, vous pouvez maintenant procéder au téléchargement et à l'installation de ControlZ :

  1. Télécharger ControlZ :
  • Téléchargez le dossier compressé ‘zip’ fourni pour obtenir le dernier exécutable de ControlZ.
  • Enregistrez le fichier sur votre ordinateur.
  1. Installer ControlZ :
  • Localisez le dossier compressé téléchargé et extrayez-le dans un dossier.
  1. Lancer ControlZ :
  • Une fois la décompression terminée, vous pouvez lancer ControlZ en double-cliquant sur le fichier ‘exe’.

Premiers pas avec ControlZ

Après le lancement de ControlZ, vous serez accueilli par le menu principal qui vous propose différentes options pour gérer vos sauvegardes. Prenez un moment pour vous familiariser avec l'interface et les différentes options disponibles.

Maintenant que vous êtes prêt, passons à la suite et découvrons comment utiliser ControlZ pour sécuriser vos données importantes.

Utilisation de l'application

Maintenant que vous avez installé avec succès ControlZ et que vous êtes prêt à démarrer, ce chapitre vous guidera à travers les différentes fonctions de l'application. Vous apprendrez à créer, modifier, exécuter et gérer vos travaux de sauvegarde.

Le Menu

Pour sélectionner un choix, il vous suffit d’entrer le numéro qui correspond à une action.

Créer un nouveau travail de sauvegarde

Pour protéger vos données importantes, vous pouvez configurer un nouveau travail de sauvegarde en suivant ces étapes :

  • Sélectionnez ‘<1> Créer un nouveau travail de sauvegarde’ à partir du menu principal, en entrant ‘1’.
  • Saisissez un nom pour le travail de sauvegarde lorsque vous y êtes invité.
  • Indiquez le chemin du répertoire source où se trouvent les données à sauvegarder.
  • Indiquez le chemin du répertoire cible où les données sauvegardées seront stockées.
  • Choisissez le type de sauvegarde : complète ou différentielle.

Visualisation des Logs

ControlZ enregistre les détails des opérations de sauvegarde dans des fichiers log pour vous permettre de suivre l'activité de l'application et de diagnostiquer d'éventuels problèmes. Ces fichiers log peuvent être particulièrement utiles pour comprendre le déroulement des sauvegardes ou pour résoudre les erreurs.

Emplacement des fichiers Log

Les fichiers log de ControlZ sont stockés dans le dossier source de l'application. Pour accéder aux logs :

  • Naviguez jusqu'au dossier d'installation de ControlZ sur votre ordinateur.
  • Recherchez le dossier nommé Logs.
  • Ouvrez ce dossier pour trouver les fichiers log, organisés par date.

Structure des fichiers Log

Chaque fichier log est nommé selon la date de l'opération de sauvegarde, suivant le format ‘YYYY-MM-DD.json’. À l'intérieur de chaque fichier, vous trouverez des informations détaillées sur les tâches de sauvegarde exécutées, y compris :

  • L'heure de début et de fin de chaque sauvegarde.
  • Les noms des travaux de sauvegarde exécutés.
  • Les chemins des répertoires source et cible.
  • Le type de sauvegarde (complète ou différentielle).
  • Les tailles des fichiers déplacés.
  • Les temps de transfère des fichiers.
